# 🎓 Student Management System (Python)

**Author:** Anuj Yadav

A Student Management System developed using **Python** that allows users to manage student records using both a **Console Interface** and a **Tkinter GUI Interface**.

The project demonstrates practical implementation of Python fundamentals, file handling, and GUI development.

---

## 🚀 Project Overview

This application performs complete student record management using **CSV file storage**.

### ✅ Features

* Add Student
* View All Students
* Search Student
* Update Student Details
* Delete Student Record
* Permanent Data Storage using CSV files

---

## 🖥️ Project Versions

### 🔹 Console Version

Menu-driven terminal application.

**Folder**

```
console_version/
```

**Run**

```bash
python main.py
```

**Concepts Used**

* Python Functions
* CSV File Handling
* CRUD Operations
* Menu-driven Programs

---

### 🔹 Tkinter GUI Version

Graphical version of the same system using **Tkinter**.

**Folder**

```
Tkinter_GUI_version/
```

**Features**

* Button-based interface
* User-friendly design
* Form inputs for student data
* Same backend logic as console version

**Run**

```bash
python main.py
```
